Проект запускается на UE 5.5;
Обратите внимание, что в проекте присутствуют кастомные классы, которые я писал на C++, поэтому, 
если у вас на машине не установлены средства компиляции кода для UE, проект может не собраться.

--------------------------------------------------
КНОПКИ УПРАВЛЕНИЯ

WASD - передвижение
SHIFT - спринт
CTRL - присяд
LMB - выстрел
RMB - прицел
Mouse Wheel - переключение оружия
--------------------------------------------------

ЧТО БЫЛО РЕАЛИЗОВАНО МНОЙ В РАМКАХ ТЕСТОВОГО ЗАДАНИЯ

Анимация:
- Написал на C++ кастомный AnimInstance для Animation Blueprint (ABP), в котором реализовал функции для получения информации о различных базовых параметрах,
  необходимых для настройки работы анимаций;
- Нашел в свободном доступе, заретаргетил и отредактировал все необходимые анимации;
- Создал и настроил все BlendSpaces, AimOffsets и Montages;
- Настроил взаимодействие State Machines в ABP:
	- Locomotion-стейты для ходьбы, присяда и спринта;
	- Промежуточные стейты (резкая остановка при беге, переход из стояния в присяд и т.п.);
	- AimOffset-стейты для прицеливания и следования головы за камерой;
	- Leaning (крен?) персонажа при движении не по прямой;
	- Рандомизация Idle-анимаций;
- Настроил IK-ноги, чтобы ступни следовали рельефу местности, а не висели в воздухе на наклонных поверхностях (контрол-риг для FootIK взял готовый);
- Настроил IK-руки, чтобы они всегда лежили на оружии;
- Настроил InPlace-повороты персонажа (когда персонаж доворачивает тело за камерой, если одного поворота головы недостаточно);

Другие системы:
- Написал на C++ кастомный класс character'a;
- Написал на C++ компонент оружия для взаимодействия character'a с оружием;
- Написал на C++ кастомный класс оружия;
- Настроил управление персонажем через EnhancedInput-систему;
- Взял готовые SkeletalMeshes и VFX-штуки для персонажа, оружия и т.п.
- Добавил простенькие элементы HUD;
